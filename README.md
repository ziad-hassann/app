# The Meg Academy Registration

Static HTML/CSS/JS registration form for GitHub Pages.

## Website Files

- `index.html`
- `admin.html`
- `styles.css`
- `config.js`
- `shared.js`
- `register.js`
- `admin.js`
- `assets/`
- `vendor/`
- `apps-script/`

## Important

To collect registrations from different devices in one shared place, deploy `apps-script/Code.gs` as a Google Apps Script Web App and paste the deployment URL into `config.js`.
